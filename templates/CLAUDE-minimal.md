# Project Configuration for Claude Code

## Workflow

1. Plan first (shift+tab twice for Plan mode)
2. Implement with verification
3. Use `/commit-push-pr` when done

## Verification

Before committing, run:
- Lint: `npm run lint`
- Types: `npm run typecheck`
- Tests: `npm test`
- Build: `npm run build`

## Standards

- Keep code simple
- No secrets in code
- Descriptive commit messages
